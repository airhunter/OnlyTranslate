# “只译” (OnlyTranslate) 隐私权政策

最后更新日期：2026年4月

“只译” 致力于保护您的隐私。本隐私政策解释了我们的浏览器扩展程序如何处理您的数据。

**1. 我们收集哪些数据？**
我们**不**收集、存储或向我们的任何服务器发送您的任何个人信息、浏览历史或网络数据。

**2. 核心数据（网页文本）的处理方式**
为了实现核心翻译功能，当您主动发起翻译请求时，插件会读取您当前网页的文本或视频字幕。这些数据会被直接发送到**您个人配置的第三方翻译 API 接口**（如 OpenAI、DeepL 等）。我们（插件开发者）不会中转、拦截或保存任何翻译内容。

**3. 本地存储**
您的配置信息（如：API 密钥、偏好语言等）仅安全地保存在您浏览器的本地存储 (`chrome.storage.local`) 中，绝不会上传至任何云端服务器。

**4. 第三方服务**
由于您的翻译数据直接发送至您配置的第三方 API 服务器，该数据的处理将受该第三方服务提供商的隐私政策约束。
